# ParticleEffectsPlus
Particle Effects+ addon for Garry's Mod
https://steamcommunity.com/sharedfiles/filedetails/?id=3684885115

Ground-up recreation of the Advanced Particle Controller addon, with the main focus being:
1. to make use of the CNewParticleEffect object and all its functionality (which didn't exist when the old addon was made)
2. to automatically read mounted .pcf files, getting a list of all effects and what functionality they have (instead of having to manually write and update particle lists, which was very tedious and made long-term maintenance untenable)
3. to seamlessly integrate particle effects as spawnmenu objects (to do away with the limited in-tool particle browser, and make use of existing spawnmenu quality-of-life features it lacked, like search, custom spawnlists, and visual spawnicons)
